# Chinmay Mittal's Portfolio

A personal portfolio website showcasing my skills, projects, and experience as a developer.

## Description

This is a static website built with HTML, CSS, and JavaScript. It features an introduction, skills section with technology logos, a projects gallery with links to my work, and a contact footer with social media links.

## Technologies Used

- HTML5
- CSS3
- JavaScript (with Typed.js library for text animation)

## Features

- Responsive navigation bar
- Animated typing effect for the introduction
- Skills showcase with icons
- Project cards linking to GitHub repositories and live demos
- Social media and contact links in the footer

## How to View

1. Clone or download the repository.
2. Open `index.html` in any modern web browser.

No server setup is required as this is a static site.
